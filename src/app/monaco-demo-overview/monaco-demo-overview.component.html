Monaco Text Editor

This game uses the Monaco text editor. This is the editor that is used in the popular IDE VS Code. This editor is very powerful and contains all kinds of cool functionality out of the box. The editor is extended with functionality like syntax highlighting and auto-complete for the language used in this game.

You might or might not be familiar with Monaco or VS Code. To make sure you can get the most out of the editor and make the experience of writing code for this game as pleasant as possible, this page shows you some tips and tricks. Monaco has a huge amount of features, which are all available to help you code. It would be a lot of work to show all the functionality. This page only shows you the most important features you will most certainly use when playing this game.
